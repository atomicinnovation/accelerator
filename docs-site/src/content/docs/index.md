Accelerator is a Claude Code plugin that structures development into discrete
phases — research, plan, implement — that communicate through the filesystem
rather than the conversation.

- [Philosophy](philosophy.md) — why the workflow is split into filesystem-linked
  phases.
- [Full Workflow](workflow.md) — how the skill families fit together end to end.
- [Development Loop](development-loop.md) — the research → plan → implement spine.
