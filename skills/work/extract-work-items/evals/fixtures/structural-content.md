# Current Issues

The notification system currently sends emails for every event with no
user-configurable preferences, leading to inbox overload for power users.

---

# Proposed Solutions

Users need a notification preferences page where they can enable or disable
email notifications per event category. Categories include: order updates,
account security, marketing, and weekly digest. Preferences should persist
to the user's profile and take effect within one hour of being saved.
