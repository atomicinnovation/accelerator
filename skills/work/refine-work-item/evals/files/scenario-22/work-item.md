# 0075: Add Dark Mode Support

**Type**: Story
**Status**: Draft
**Priority**: Low
**Author**: Test Author

## Summary

As a user, I want to switch to dark mode so that I can use the app comfortably in low-light environments.

## Context

Dark mode is a frequently requested feature. The current UI uses a fixed light theme.

## Requirements

1. Add a dark mode theme to the design system.
2. Allow users to toggle dark mode from their settings.
3. Persist the user's theme preference.

## Acceptance Criteria

- [ ] Given a user toggles dark mode, all UI elements switch to the dark theme within 100ms.
- [ ] Given a user reloads the page, the previously selected theme is restored.
