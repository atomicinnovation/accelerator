# Apply case-insensitively (-iE) for the word-variation patterns below.
# The implement pattern is applied case-sensitively (grep -E, no -i) so that
# "implement Foo" (proper noun) matches but "implement foo" does not.
#
# These patterns mirror the cue-phrase enumeration in
# skills/work/extract-work-items/SKILL.md:130-138.

# Note: applied with case-sensitive grep — requires an uppercase letter
# immediately after "implement " to identify proper-noun feature names.
# Use [Ii]mplement [A-Z] to allow sentence-case "Implement Foo" while
# rejecting lowercase "implement foo". Do not apply this pattern with -i.
