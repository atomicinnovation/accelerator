# 0141: Effort Control for Review Skills

**Kind**: Story
**Status**: Draft
**Priority**: Medium
**Author**: Toby Clemson

## Summary

Consider adding effort control to review skills, letting users dial how
thorough (and expensive) a review is.

## Context

Extracted from the ideas backlog note
(`meta/notes/2026-06-22-ideas-backlog.md`). Framed as a consideration. Reviews
can vary widely in depth; an effort control would make that explicit.

## Requirements

- Add an effort/depth control to review skills.
- Define the levels and how they scale review thoroughness.

## Acceptance Criteria

- [ ] Review skills accept an effort level that visibly changes review depth /
      breadth.

## Open Questions

  reasoning depth?
