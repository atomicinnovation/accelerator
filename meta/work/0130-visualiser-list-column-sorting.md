# 0130: Sort Visualiser Lists by Column Headers

**Kind**: Story
**Status**: Draft
**Priority**: Medium
**Author**: Toby Clemson

## Summary

Allow sorting visualiser list views by clicking column headers, so users can
order artefacts by any displayed column.

## Context

Extracted from the ideas backlog note
(`meta/notes/2026-06-22-ideas-backlog.md`). Visualiser list views display
artefacts in columns but do not currently support interactive sorting.

## Requirements

- Make column headers in list views interactive for sorting.
- Support ascending/descending toggle and a clear indication of the active sort.

## Acceptance Criteria

- [ ] Given a list view, when the user clicks a column header, then the list
      reorders by that column and the sort direction is indicated.

## Open Questions

- Should the active sort persist across navigation / sessions?

## Dependencies

- Blocked by: None identified yet.
- Blocks: None identified yet.

## Assumptions

- Applies to all tabular list views in the visualiser.
